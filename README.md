File naming convention:
Audible or Control
Program 1 or Program 2 (If it was a regular IPG program used by the patient)
R or L (Right or Left) GPi or STN (Globus Pallidus internus or Subthalamic nucleus) 
Contacts stimulated (Schematic available in paper) 
Whether they had high impedance or normal impedance readings 
Total current amplitude delivered 
Pulse frequency
All separated by underscores
